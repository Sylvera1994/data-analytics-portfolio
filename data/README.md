# Dataset: World Happiness Report (2005-2021)

This dataset contains annual scores for countries around the world measuring factors contributing to happiness.  
Columns include:

- Country
- Year
- Happiness Score
- GDP per capita
- Social support
- Healthy life expectancy
- Freedom
- Generosity
- Corruption perceptions

This dataset will be used for hands-on analysis in SQL, Excel, Tableau, and Python.
